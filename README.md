# Simple-Tic-Tac-Toe-Game
This is a Simple Tic Tac Toe(O X) Game made using python 

Output as given below:-

0 | 1 | 2\n
3 | 4 | 5\n
6 | 7 | 8\n
X's turn\n
Choose a position from 0-8: 1\n
0 | X | 2/n
3 | 4 | 5\n
6 | 7 | 8\n
O's turn\n
Choose a position from 0-8: 4\n
0 | X | 2\n
3 | O | 5\n
6 | 7 | 8\n
X's turn\n
\n
Also don't get confused by "0" and "O".
